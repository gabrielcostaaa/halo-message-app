# Pasta de Avatares

Coloque as imagens dos avatares dos usuários aqui.

## Nomenclatura:

- `alice.jpg`
- `bruno.jpg`
- `carlos.jpg`
- `diana.jpg`
- `eduardo.jpg`

## Formatos suportados:
- JPG/JPEG
- PNG
- WebP

## Recomendações:
- Tamanho: 200x200px ou 400x400px
- Peso máximo: 500KB por imagem
- Imagens quadradas

## Acesso:
As imagens estarão disponíveis em:
`http://localhost:3000/avatars/nome-do-arquivo.jpg`
